# Настройка автоматического обновления через GitHub для Vvyiloff/Post-Install

## Как это работает

Программа уже настроена для работы с вашим репозиторием `Vvyiloff/Post-Install`. Теперь вам нужно разместить файл `packages.json` в вашем репозитории, чтобы программа могла загружать обновления.

## Шаг 1: Разместите файл в вашем репозитории

1. Скопируйте файл `packages.json` из этой папки (`casino-app/packages.json`)
2. Перейдите в ваш репозиторий: [https://github.com/Vvyiloff/Post-Install](https://github.com/Vvyiloff/Post-Install)
3. Нажмите "Add file" -> "Upload files"
4. Загрузите файл `packages.json` в корень репозитория
5. Нажмите "Commit changes"

## Шаг 2: Использование

Теперь программа будет проверять обновления из вашего репозитория:

1. Запустите программу
2. Нажмите кнопку "Проверить обновления" на вкладке "Система"
3. Если вы обновите файл `packages.json` в репозитории, программа обнаружит изменения
4. Пользователям будет предложено обновить список программ
5. После обновления список программ автоматически перезагрузится

## Формат файла packages.json

```json
[
    {
        "name": "Название программы",
        "id": "Winget Package ID",
        "group": "Категория",
        "reboot": true  // Optional: требуется ли перезагрузка после установки
    }
]
```

## Пример рабочего процесса

1. Вы добавляете новую программу в `packages.json` в вашем репозитории
2. Делаете коммит и пуш в GitHub
3. Пользователи запускают вашу программу и нажимают "Проверить обновления"
4. Программа обнаруживает изменения и предлагает обновить список
5. После обновления пользователи видят новую программу в списке

## Как добавить новую программу

1. Найдите правильный Winget Package ID для программы
2. Добавьте новый объект в массив в `packages.json`
3. Укажите название, ID, группу и при необходимости флаг перезагрузки
4. Сохраните файл и загрузите в репозиторий

## Важно

- Репозиторий должен быть публичным (ваш репозиторий уже публичный)
- Файл должен называться именно `packages.json` и находиться в корне репозитория
- Формат JSON должен быть корректным
- Убедитесь, что все package ID корректны и доступны через winget

## Проверка работы

После того как вы разместите файл в репозитории, вы можете проверить работу системы обновлений:

1. Запустите программу
2. Нажмите "Проверить обновления"
3. Если всё настроено правильно, программа загрузит список из вашего репозитория
