# Changelog

Все значимые изменения в этом проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.1] - 2024-XX-XX

### Исправлено
- Исправлены кнопки управления окном (свернуть, развернуть, закрыть)
- Улучшена стабильность IPC коммуникации между процессами

### Изменено
- Обновлена документация
- Улучшена структура проекта

## [1.0.0] - 2024-XX-XX

### Добавлено
- Первый релиз проекта
- Python версия установщика (консольная)
- Electron версия установщика (графический интерфейс)
- Поддержка установки программ через winget
- Автоматическая проверка установленных программ
- Профили быстрой установки (Игры, Разработка, Базовый софт)
- Управление DNS настройками
- Система логирования с фильтрацией
- Темная и светлая тема (Electron версия)
- Поиск и фильтрация программ
