# Software Installer

Универсальный установщик программ с поддержкой автоматического обновления через GitHub.

## Описание

Этот установщик позволяет легко устанавливать и управлять программами через winget с возможностью автоматического обновления списка доступных программ из GitHub репозитория.

## Особенности

- ✅ Установка программ через winget
- ✅ Автоматическое обновление списка программ из GitHub
- ✅ Управление установленными программами (удаление)
- ✅ Фильтрация программ по категориям
- ✅ Показ статуса установки в реальном времени
- ✅ Быстрый запуск благодаря отложенной загрузке

## Требования

- Windows 10/11
- Установленный winget
- Интернет-соединение для проверки обновлений

## Установка

1. Склонируйте репозиторий
2. Перейдите в папку 
3. Запустите `software_installer.py`

```bash
python software_installer.py
```

## Использование

1. **Установка программ**:
   - Выберите нужные программы из списка
   - Нажмите "Начать установку"
   - Следите за прогрессом в реальном времени

2. **Управление установленными программами**:
   - Перейдите на вкладку "Установлено"
   - Двойной клик по программе для удаления

3. **Проверка обновлений**:
   - Нажмите "Проверить обновления" на вкладке "Система"
   - Подтвердите обновление списка программ

## Настройка GitHub

Программа настроена для работы с репозиторием `Vvyiloff/Post-Install`. Для настройки:

1. Разместите файл `packages.json` в корне вашего репозитория
2. Программа будет автоматически загружать обновления

См. [GITHUB_SETUP.md](GITHUB_SETUP.md) для подробной инструкции.
